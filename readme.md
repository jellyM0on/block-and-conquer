## Block&Conquer 

### Commits 
**Format**: (type)(scope): (message in  imperative present tense)

**Types**: 
- feat - adds/removes a feature
- fix - fixes a bug
- refactor - rewrites/restructures code 
- style 
- test - test related 
- docs - documentation
- build - project build components
- chore 

**Reference**: https://www.conventionalcommits.org/en/v1.0.0/
